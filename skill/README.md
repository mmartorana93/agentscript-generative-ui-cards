# Skill: agentscript-generative-ui-cards

A Claude Code skill that recreates the marker-based generative-UI card pattern on any Agent Script
agent + custom chat. See [`SKILL.md`](SKILL.md).

## Install (Claude Code)

```bash
mkdir -p ~/.claude/skills/agentscript-generative-ui-cards && \
  curl -fsSL https://raw.githubusercontent.com/mmartorana93/agentscript-generative-ui-cards/main/skill/SKILL.md \
    -o ~/.claude/skills/agentscript-generative-ui-cards/SKILL.md
```

Then invoke it by describing the goal ("add graphic cards to my custom Agentforce chat"), or with
`/agentscript-generative-ui-cards`.
